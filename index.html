게임 계정을 삭제하는 방법

게임 내 설정 메뉴에서 직접 계정을 삭제할 수 있습니다.

계정 삭제 절차

  1. 게임을 실행합니다.
  2. 오른쪽 상단에 있는 설정 버튼을 누릅니다.
  3. 계정 삭제 메뉴를 선택합니다.
  4. 화면에 표시된 안내에 따라 삭제를 확인합니다.

삭제되는 정보

  • 플레이어 진행 데이터
  • 인벤토리 및 구매 아이템
  • 서버 로그(접속 기록, 오류 로그 등)
  • 서버에 저장된 모든 계정 관련 데이터

보관 기간

계정 삭제 요청은 15일 대기 상태에 들어갑니다. 이 기간 동안 요청을 철회하고 싶다면 문의를 통해 취소할 수 있습니다.

15일의 유예기간이 지나면 모든 관련 데이터는 영구적으로 삭제되며 복구가 불가능합니다.
